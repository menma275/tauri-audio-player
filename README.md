# Antigravity Music Player (Tauri + React + TypeScript)

Tauri 2.0、React、TypeScript、Vite、Tailwind CSS v4を使用したローカルミュージックプレイヤー。

---

## 🛠 機能一覧

### 1. テーマ切り替えとOS連携
*   **ダーク/ライトモードのトグル**: ダークモードとライトモードの切り替えに対応。設定はローカルに保存され、起動時に自動で復元される。
*   **OSタイトルバー同期**: アプリ内のテーマ変更に合わせて、OS標準のウィンドウタイトルバーとボタンの色調も自動的に連動して切り替わる。

### 2. インライン・分割ドロワーレイアウト
*   **可変レイアウト比率**: ライブラリ（左サイドバー）と歌詞表示（右パネル）の開閉状態に応じて、領域比率が自動で調整される。
    *   両方展開時： **1 : 3 : 1** の比率に分割
    *   片方展開時： 展開側と中央カバー領域が **1 : 4** の比率に分割
    *   両方格納時： 中央カバー領域が **100%** を占有
*   **個別ドラッグリサイズ**: 左右ドロワーの境界線をドラッグすることで幅を調整可能。
    *   最小幅 `190px` から最大幅 `400px` の間で制限
    *   リサイズ操作中はアニメーションを一時的に無効化し、ドラッグ時の追従性を向上
    *   調整した幅はローカルに保存される

### 3. ミニマルフォルダセレクター
*   **操作と表示の統合**: フォルダ選択のアクションボタンと、選択中のフォルダ名表示を1つのコンパクトなボタン要素に統合。クリックすることで別フォルダへの切り替えができる。

### 4. キーボードショートカット
*   **スペースキー**: 再生 / 一時停止のトグル切り替え。
    *   ライブラリ検索などのテキスト入力欄にフォーカスがある間は、ショートカットが無効化されスペース入力が優先される。
    *   ショートカット実行時のブラウザ標準のスクロール挙動を防止。

## 🚀 インストールと起動

### 準備
*   Node.js (v18以降)
*   Rust ツールチェーン (Tauriビルド用)

### インストール
```bash
pnpm install
```

### 開発用サーバー起動
```bash
pnpm tauri dev
```

### プロダクションビルド
```bash
pnpm tauri build
```

## 📁 ディレクトリ構成

*   `src/`: フロントエンドコード (React)
    *   `App.tsx`: アプリケーションのメインロジック、ショートカット、描画DOMの制御
    *   `App.css`: Tailwindテーマ変数、スクロールバー制御
    *   `main.tsx`: エントリーポイント
*   `src-tauri/`: バックエンドコード (Tauri / Rust)
    *   `src/main.rs`: フォルダ内スキャン、LRC歌詞ファイルの読み込み処理
    *   `capabilities/default.json`: ウィンドウテーマ制御権限の許可設定
    *   `tauri.conf.json`: ウィンドウサイズやOSデコレーションの設定
